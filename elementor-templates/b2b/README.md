# B2B trade templates (Wholesale)

## Templates Included

1. **Product Detail**
   - File: `01-product-detail-nova/b2b-product-detail-nova.json`
   - Purpose: Trade product page with spec grid, quote CTA, related compatibilities
   - URL: `/professionals/products/{id}` (align permalinks in WordPress)
   - Setup: 15–20 min per product (image, copy, spec grid HTML)

2. **About Us**
   - File: `02-about-us-nova/b2b-about-us-nova.json`
   - Purpose: Trade positioning — feature blocks and proven deployments
   - URL: `/professionals/about`
   - Setup: 10–15 min

3. **Contact Us**
   - File: `03-contact-us-nova/b2b-contact-us-nova.json`
   - Purpose: Trade access requests (Elementor **Form** widget)
   - URL: `/professionals/contact`
   - Setup: 10–15 min

## How to Import

1. In WordPress: **Pages → Add New**
2. **Edit with Elementor**
3. **Templates → Insert Template**
4. Upload the JSON file
5. Click **Insert**
6. Map forms and links to your trade portal URLs, then publish

## Customization

- Spec grid is an **HTML** widget; adjust per SKU or replace with native widgets
- Links use `/professionals/...` — update to match your site structure
- See `../IMPORT_INSTRUCTIONS.md` for global import notes
